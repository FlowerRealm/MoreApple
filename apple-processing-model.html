More Apple 加工流程模型

本轮实现加工核心能力，不实现调用入口。配方、GUI、命令、匠魂台以后都可以调用同一个服务。

核心结构

moreapple:apple
└─ MoreApple
   ├─ SchemaVersion: 1
   └─ Modifiers
      ├─ { Id: "moreapple:golden", Level: 1 }
      └─ { Id: "moreapple:healing", Level: 2 }
没有“最多 3 个 upgrade”限制。底层只限制单个属性自己的 maxLevel 和互斥组。

字段与取舍

字段用途取舍
id属性身份。支持扩展；必须处理未知 id。
level属性等级。升级改等级，不重复塞多个同名属性。
maxLevel单属性上限。限制单属性，不限制总属性数。
exclusiveGroup互斥组。把冲突规则数据化，少写 if/else。
priority未来显示和结算顺序。稳定排序，别靠 NBT 顺序碰运气。
